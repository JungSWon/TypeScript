##  Visual Studio Code 

###  Compiler 
- VSCode에 컴파일러가 내장되어있음
- 내장된 컴파일러 버전은 VSCode 업데이트시 함께 업데이트
- 그래서 컴파일러 버전과 VSCode 버전은 상관 관계가 있음
- 내장된 컴파일러 or 직접 설치한 컴파일러 선택가능
